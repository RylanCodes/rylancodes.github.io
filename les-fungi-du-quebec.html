The Québec Fungi Community

My final school project was creating a website for mushroom enthusiasts to connect and share their love for fungi.

    Stack
  • Vue.js
  • Javascript
  • HTML
  • CSS
  • MongoDB

Project Purpose & Goal

The goal was to consolidate our knowledge by building a functional website for a potential company. We aimed for reusable and accessible code, utilized the Atlassian suite, created a wireframe on Figma, and wrote tests to ensure functionality. Additionally, I challenged myself to use pure CSS instead of a CSS framework.

Challenges & Thought Process

We learned Quasar but had to pivot due to unit test conflicts. Working with 4 unfamiliar classmates taught us the importance of focusing on strengths and effective communication. We encountered issues with downloading images to the server and learned how to use AWS. We conducted research to create a wireframe, established a good coding structure from the start, and reused classes to save time.

Lessons Learned

This project taught me to choose my battles and the importance of collaboration and communication in a team setting. I gained more experience in frontend and backend development and the value of writing reusable code. Overall, the project was a positive learning experience that challenged me to grow as a developer.

Scarlatta Fortuna

For my second-to-last school project, I collaborated with two fellow students to create a haute couture e-commerce website. As the designated designer of the team, my teammates entrusted me with the task of the frontend code and the overall appearance of the site.

    Stack
  • Vue.js
  • Javascript
  • HTML
  • CSS
  • MongoDB

Project Purpose & Goal

Our primary goal was to learn how to incorporate Vue.js into our project and to become more proficient with Git and Figma. I challenged myself to create the most beautifully designed website in the class.

Challenges & Thought Process

Designing a beautiful project without any design experience was a new experience for all of us. We also had to learn how to use a framework at the same time, which added stress and led to some mistakes with our Git branches. However, I am proud of the efforts we put into searching for website inspirations, color palettes, and fonts, and how we communicated throughout the process to ensure we were on the same page.

Lessons Learned

This project taught us not to take ourselves too seriously and to have backup plans, especially when we make mistakes. We also learned the importance of planning from the beginning, reusing our code, and making sure it is responsive to avoid refactoring later.

My Hiking Trails

For my first project using JavaScript, I was given the freedom to choose the theme, but had to adhere to specific requirements and strict corrections from my teacher.

    Stack
  • HTML

Project Purpose & Goal

The main goal of the project was to learn the basics of vanilla JavaScript and understand how to use it in a coding project.

Challenges & Thought Process

I had to learn how to manipulate the DOM, use classes, functions, methods, and constructors. During challenging learning moments, I would remind myself of the quote, 'Learning to code is like learning every single part of a car individually but having to teach yourself how to put it together'. I watched many YouTube tutorials to help me understand coding concepts.

Lessons Learned

Through building a project from scratch, I learned to focus on one step at a time to avoid being overwhelmed. Breaking down the task and tackling the part that I could do first helped. I also learned to not hesitate to reach out to colleagues for help and ask questions to the teacher.
